## **Personal Expense Tracker**

This is a web-based **Expense Tracker** built using **Django** that helps users track and monitor their daily expenses.

## Features

- Sign Up / Login page
- Set monthly income
- Set expense limit
-  Add and delete expenses
- Warning when expenses exceed limit
- Category-wise expense tracking
- Remaining balance calculation
- Django admin panel for database management

## Tech Stack

- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite Database
